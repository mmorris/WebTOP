getGeometricProperty

public static org.web3d.x3d.sai.X3DField getGeometricProperty(org.webtop.x3d.NamedNode geometry,
                                                              java.lang.String property)
Returns the X3DField for the geometric property property of the geometry node contained in geometry cast as it's appropriate type:
  • MFColor if property = "color".
  • MFVec3f if property = "coord" or "normal".
  • MFVec2f if property = "texCoord".
Parameters:
geometry - NamedNode containing the geometry X3DNode
property - String containing the property name
Returns:
X3DField of the property cast as it's appropriate type.
