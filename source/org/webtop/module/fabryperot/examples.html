Interference, Fabry-Perot Etalon: Examples

To view an example, press the button at the end of the description. Once the module finishes loading, press the play button at the recording panel. If you want to save the example code to your computer and run it locally right click of the disk icon and save the file to disk. Use wsl as an extension of the file you save to your computer.

Example 1  Changing the Reflectivity

In this example light from a 550 nm point source is incident upon a slab of glass (nG = 1.50) that is 1.00 cm thick and is coated on both sides. The reflectivity of each face is R. The intensity of the light transmitted through the slab is viewed on an observation screen. The following cases are depicted consecutively:
(a) R = 0.3,
(b) R = 0.5,
(c) R = 0.7,
(d) R = 0.9.

Example 2 Changing the Thickness

In this example light from a 550 nm point source is incident upon a slab of glass (nG = 1.50) of thickness d that is coated on both sides. The reflectivity of each face is 0.70. The intensity of the light transmitted through the slab is viewed on an observation screen. The following cases are depicted consecutively:
(a) d = 0.50,
(b) d = 0.75,
(c) d = 1.00,
(d) d = 1.25.

Example 3 Changing the Wavelength

In this example light from monochromatic point source of wavelength l is incident upon a slab of glass (nG = 1.50) that is 1.00 cm thick and is coated on both sides. The reflectivity of each face is 0.70. The intensity of the light transmitted through the slab is viewed on an observation screen. The following cases are depicted consecutively:
(a) l = 400 nm,
(b) l = 500 nm,
(c) l = 600 nm,
(d) l= 700 nm.
