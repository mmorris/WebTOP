Exercise 1:  Varying the Wavelength of a Spherical Wave

Starting from the default configuration, use the Wavelength number box to set the wavelength of the incident light to 400 nm. Use the Wavelength number box to set the wavelength of the incident light to 800 nm. How does this affect the spacing of the wavefronts?

 

Exercise 2:  Varying the Wavelength of a Plane Wave

Starting from the default configuration, choose the Plane Wave flap and use the Wavelength number box to set the wavelength of the incident light to 400 nm. Use the Wavelength number box to set the wavelength of the incident light to 800 nm. How does this affect the spacing of the wavefronts?

 

Exercise 3:  Varying the Direction of Propagation of a Plane Wave.

Start from the default configuration, choose the Plane Wave flap. Use the Rotate navigation icon to make the plane wave travel in the following directions: (a) towards the right of your computer screen, (b) towards the top of your computer screen, and (c) into your computer screen.
