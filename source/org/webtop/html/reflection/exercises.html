Comments

There are two different types of fields available to the user: polarized or unpolarized. The user chooses the type by clicking on either the Polarized or Unpolarized tab in the upper left hand corner of the Console.

In the exercises below, the user may want to use the stop button to determine certain properties of the waves, e.g., whether or not a phase change upon reflection has occured.

 

Exercise 1:  s-Polarized Light Incident from Air onto Glass

Set the Incident field type to Polarized. Set the index of refraction of the incident medium to be 1.00 and the index of refraction of the transmitting medium to be 1.50. Set the value of Lambdai to 550 nm, the value of Eperp to 2.60, and the value of Epar to zero. The value of Phase Diff. is irrelevant, since only one component is nonzero. Set the display mode to Perpendicular.

(a) Set value of the angle of incidence to 25 degrees.

      (i) Is there a p phase change upon reflection? Use the relevant Fresnel equation to verify your answer.

      (ii) Is there a p phase change upon transmission? Use the relevant Fresnel equation to verify your answer.

      (iii) Which field, the reflected or the transmitted, has the larger amplitude? Use the relevant Fresnel equations to verify your answer.

      (iv) Is the wavelength of the light in the second medium smaller than or larger than the wavelength in the first medium?

(b) Set value of the angle of incidence to 89.99 degrees.

      (i) Is there a p phase change upon reflection? Use the relevant Fresnel equation to verify your answer.

      (ii) Approximately what percentage of the incident intensity is reflected, and what percentage is transmitted?

Exercise 3:  Linearly Polarized Light Incident from Air onto Glass

Set the Incident field type to Polarized. Set the index of refraction of the incident medium to be 1.00 and the index of refraction of the transmitting medium to be 1.50. Set value of Lambdai to 550 nm, the value of Eperp to 2.60, the value of Epar to 2.60, and the value of Phase Diff to zero. Set the display mode to Composite. The incident electric field is linearly polarized; it makes an angle of 45 degrees with respect to the plane of incidence. Brewster’s angle for this situation is 56.31 degrees. In order to answer the questions below, you will need to use your mouse to rotate the scene so that you can determine the polarization of the electric field in question (the incident or reflected or transmitted).

(a) Set value of the angle of incidence to 25 degrees. 

      (i) The reflected electric field is also linearly polarized. Is it polarized in approximately the same direction as the incident field, or in a different direction?

      (ii) The transmitted electric field is also linearly polarized. Is it polarized in approximately the same direction as the incident field, or in a different direction?

(b) Set value of the angle of incidence to 56.31 degrees.

       (i) What is the polarization of the reflected field?

       (ii) What is the polarization of the transmitted field?

(c) Set value of the angle of incidence to 89.99 degrees. What is the polarization of the reflected field?

Set the display mode to Composite. Set the angle of incidence to be 56.31 degrees. Set the display mode to Composite. What is the polarization of the reflected field?

 

Exercise 5:  s-Polarized Light Incident from Glass onto Air

Set the Incident field type to Polarized. Set the index of refraction of the incident medium to be 1.50 and the index of refraction of the transmitting medium to be 1.00. Set the value of Lambdai to 550 nm, the value of Eperp to 2.60, and the value of Epar to zero. The value of Phase Diff. is irrelevant, since only one component is nonzero. Set the display mode to Perpendicular. The critical angle for this situation is 41.81 degrees.

(a) Set value of the angle of incidence to 30 degrees.

      (i) Is there a p phase change upon reflection? Use the relevant Fresnel equation to verify your answer.

      (ii) Is there a p phase change upon transmission? Use the relevant Fresnel equation to verify your answer.

      (iii) It should be obvious that the amplitude of the transmitted wave is larger than the amplitude of the incident wave. Reconcile this with the law of conservation of energy. (Hint: Consider a cylinder, call it Ci, full of parallel incident rays. Use the law of refraction to draw the cylinder, call it Ct, full of refracted rays that are generated by the incident rays in Ci . Which has the larger cross sectional area?)

(b) Set value of the angle of incidence to 41.82 degrees. Warning: We are not depicting the evanescent    waves in the air.

      (i) The phase change upon reflection in this case is zero. Verify this from the module and from the relevant theory.

      (ii) What percentage of the incident intensity is reflected?

 (c) Set value of the angle of incidence to 58 degrees. Warning: We are not depicting the evanescent waves in the air.

      (i) The phase change upon reflection in this case is pi/2. Verify this from the module and from the relevant theory.

      (ii) What percentage of the incident intensity is reflected?

(d) Set value of the angle of incidence to 89.99 degrees. Warning: We are not depicting the evanescent waves in the air.

      (i) The phase change upon reflection in this case is pi. Verify this from the module and from the relevant theory.

      (ii) Approximately what percentage of the incident intensity is reflected?
