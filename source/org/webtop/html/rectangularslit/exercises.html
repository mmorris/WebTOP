Exercise 1:  Varying the Width and Height of the Aperture

(a) Starting from the default configuration, use the Aperture widget to increase the width of the aperture. How is the diffraction pattern on the observation screen affected? (b) Use the Aperture widget to increase the height of the aperture. How is the diffraction pattern on the observation screen affected?

 

Exercise 2:  Varying the Wavelength of the Incident Light

Starting from the default configuration, use the Wavelength widget to increase the wavelength of the incident light. How is the diffraction pattern on the observation screen affected?

 

Exercise 3:  Varying the Distance to the Observation Screen

Start from the default configuration. Use the "z" widget to increase the distance from the aperture plane to the observation plane. How is the diffraction pattern on the observation screen affected?
