TOP is comprised of seven modules: Wave Simulation, Reflection and Refraction, Geometrical Optics, Polarization, Interference, Fraunhofer Diffraction, and Fresnel Diffraction. A web page for each module may be accessed through the links provided in the frame on the left of this screen. In the web page for each module we have provided you with links to

1. A brief theory which relates to that module,
2. Still pictures taken from the module at work,
3. Movies created from images which were captured during a TOP session with that module.

In addition, on the Single Slit section of the Fresnel Diffraction module page, there is a link to an interactive 3D web based version of this submodule which was created using VRML 2.0 and Java.

Please note that you may need one or more of the following plugins to your web browser to view the movies/VRML models of the modules. Links to locations from where you can download these plugins are provided in the appropriate places. We recommend you to follow the links provided here to download those plugins before hand, if your browser doesn't have them already.

1. You will need QuickTime movieplayer plugin to view the movies.
2. You will need the CosmoPlayer plugin to view the VRML models.
3. In addition, your web browser should support Java scripting.
