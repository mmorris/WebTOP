Wave Simulation

This module simulates waves in a ripple tank. The user can insert up to eight point sources (which generate circular waves) or "line" sources (which generate wavefronts that are straight lines). For each circular wave the user specifies: (a) the two coordinates which specify the position of the source in the plane of the undisplaced water surface, (b) the source "strength", and (c) the wavelength. For each "line" source, the user specifies (a) the direction the wave is traveling, (b) the amplitude of the wave, (c) the initial phase of the wave, the wavelength of the wave.
