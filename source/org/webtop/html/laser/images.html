Lasers: Images

Definitions

Wavelength = laser wavelength in vacuum
R1 = radius of curvature of Mirror 1
R2 = radius of curvature of Mirror 2
L = distance between mirrors
z = distance from beam waist to observation screen



Image 1: TEM 00 Gaussian Beam
Wavelength = 500 nm, R1 = R2 = 520 mm, L = 1000 mm, and z = 2000 mm. The laser cavity is a near-concentric resonator.

To download the high resolution image, copy the following link into your web browser http://webtop.msstate.edu/images/lasers/lasers-image-1.tif



Image 2: TEM 00 Gaussian Beam
The parameter values are the same as those of Image 1, except that L = 520 mm. Since R1 = R2 = L, the laser cavity is a confocal resonator.

To download the high resolution image, copy the following link into your web browser http://webtop.msstate.edu/images/lasers/lasers-image-2.tif
