Lasers: Examples

To view an example, press the button at the end of the description. Once the module finishes loading, press the play button at the recording panel. If you want to save the example code to your computer and run it locally right click of the disk icon and save the file to disk. Use wsl as an extension of the file you save to your computer.

Example 1  Varying the Length of the Cavity

In this example, the cavity begins in a near-concentric configuration (R1 = R2 = 520 mm, L = 1000 mm). The length of the cavity is slowly decreased until the configuration is confocal (L = R1 = R2 = 520 mm). Note that as the length of the cavity decreases, the spot size at the beam waist increases, and the beam is more collimated (i.e., it diverges less).

Example 2  Varying the Radii of Curvature of the Mirrors

In this example, the cavity begins in a near-concentric configuration (R1 = R2 = 520 mm, L = 1000 mm). The radii of curvature are then increased to R1 = R2 = 1000 mm (a confocal configuration), and then to R1 = R2 = 10,000 mm (a near-planar configuration). Note that as the radii of curvature of the mirrors are increased, the spot size at the beam waist increases, and the beam is more collimated (i.e., it diverges less).

Example 3 Varying the Wavelength

In this example, the cavity is in a near-concentric configuration. The wavelength begins at the value 400 nm, and is then increased to 550 and 700 nm. The cycle is then repeated. Note that as the wavelength is increased, the spot size at the beam waist increases, and the beam is more collimated (i.e., it diverges less).

 

Exercise 4  Low Order Transverse Modes

In this example, the beam starts out in the TEM 00 transverse mode. A vertical wire is then inserted into the cavity, which forces the laser to oscillate in the TEM 10 mode. Note that this mode has one nodal line in the vertical direction and none in the horizontal direction. The vertical wire is then replaced by a horizontal one, which forces the laser to oscillate in the TEM 01 mode. Note that this mode has no nodal lines in the vertical direction and one in the horizontal direction. The horizontal wire is then replaced by a pair of crossed wires, which forces the laser to oscillate in the TEM 11 mode. Note that this mode has one nodal line in the vertical direction and one in the horizontal direction.
