Interference, Michelson Interferometer Module: Examples

To view an example, press the button at the end of the description. Once the module finishes loading, press the play button at the recording panel. If you want to save the example code to your computer and run it locally right click of the disk icon and save the file to disk. Use wsl as an extension of the file you save to your computer.

Example 1 Change of the Path Length Difference

In this example a monochromatic point source illuminates a Michelson interferometer, and the resulting interference pattern is observed on a screen.  The tilt mirror is not tilted, i.e., f = 0. Let the path length difference for the waves traveling the two different paths be denoted by d and the wavelength of the light by l. The example begins with the parameter values d = 0.007 cm, and l = 500 nm. In this case 2d/l is an integer, and, as a result, the two waves arriving at the center of the observation screen interfere destructively, making the center of the screen dark. (If you don’t agree, recall that there is a p phase difference between the two beams introduced at the beam splitter.) The value of d is then slowly increased, using the translation mirror widget, until one new fringe is introduced. Subsequently, the translation mirror keyboard entry box is used to increase the value of d to 0.008 cm, and then 0.009 cm.

Example 2 Change of the Wavelength

In this example a monochromatic point source illuminates a Michelson interferometer, and the resulting interference pattern is observed on a screen.  The tilt mirror is not tilted, i.e., f = 0. Let the path length difference between the waves traveling the two paths be denoted by d and the wavelength of the light by l. The example begins with the parameters values d = 0.007 cm, and l = 500 nm. In this case 2d/l is an integer, and as a result the two waves arriving at the center of the observation screen interfere destructively, making the center of the screen dark. The value of l is then slowly decreased, using the wavelength widget, until one new fringe is introduced. Subsequently, the wavelength keyboard entry box is used to change the value of l to 400 nm, then 500 nm, then 600 nm, and then 700 nm.

Example 3 Tilting the Tilt Mirror

In this example a monochromatic point source illuminates a Michelson interferometer, and the resulting interference pattern is observed on a screen. The path length difference for the two beams is 0.007 cm and the wavelength is 500 nm. The tilt of the tilt mirror starts at 0 degrees, and is increased, using the phi keyboard entry box, to 0.001, 0.002, 0.003, and 0.004 degrees.
