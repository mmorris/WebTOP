Exercise 1:  Changing the Path Length Difference

Set the value of the wavelength to be 500 nm, the value of the path length difference d to 0.01455 cm, and the tilt angle f to zero.

(a) Is the intensity at the center of the intensity pattern a maximum or a minimum? Is this what you expected? If not, did you consider the p phase difference between the two beams introduced at the beam splitter due to the fact that the two beams reflect from it differently?

(b) Decrease d to 0.00970 cm. Does the number of visible rings decrease or increase?

(c) Decrease d to 0.00485 cm. Does the number of visible rings decrease or increase?

(d) Place your cursor over the wheel widget on the back of the translation mirror. (The cursor will change color when it is in the proper position.) Press down your left mouse button, and rotate the wheel clockwise. Are the rings moving inwards or outwards?

 

Exercise 2:  Rotation of the Tilt Mirror

Set the value of the wavelength to be 500 nm, the value of the path length difference d to be 0.00485 cm, and the tilt angle f to be zero. Use the Phi keyboard entry box to change the angle of the tilt mirror to 0.1, 0.2, and 0.3 degrees. How does the observation pattern on the screen change? Is this what you expected?

 

Exercise 3:  Change of the Wavelength

Set the value of the wavelength to be 500 nm, the value of the path length difference to be 0.00485 cm, and the tilt angle f to be zero. Use the cursor to engage the wavelength widget (the wheel on the right hand side of the scene), and rotate the wheel slowly (either direction is fine). How does the observation pattern on the screen change? Is this what you expected?
