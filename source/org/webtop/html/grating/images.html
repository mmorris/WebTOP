Transmission Diffraction Grating: Images

Definitions N = number of grooves
w = width of a groove
d = distance between consecutive grooves
Wavelength = wavelength in vacuum of incident light

Image 1
N = 50, w = 0.100 microns, d = 2.00 microns, Wavelength 1 = 450 mm, and Wavelength 2 = 500 mm.

To download the high resolution image, copy the following link into your web browser http://webtop.msstate.edu/images/grating/grating-1.tif
