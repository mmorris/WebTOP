Scattering: Theory

Introduction

This module simulates the non-resonant scattering of light from an atom. The atom is located at the origin, and the incident light is traveling along the negative y-axis in the positive y-direction. The oscillating pink arrow depicts the induced dipole moment of the atom. The scattered electric field is depicted along three different axes: the positive x-axis, the positive z-axis, and movable axis whose angular position the user can vary (see Fig. 1 below). The spherical polar angles q and f are used to specify the position of the movable axis. The module has two options for the incident field: Polarized (Linearly) and Unpolarized.

Figure 1 The Scattering module. The incident field is linearly polarized in the x-z plane, and the movable observation screen is located at q = 90 degrees and f = 180 degrees.

When the incident field is polarized, there are five parameters that the user can vary: E0x, E0z, l, q, and f. q, and f are, respectively, the polar and azimuthal angles of the movable observation screen. Each of these parameters can be varied either by using the appropriate keyboard entry box on the Console, or by using the appropriate widget in the scene. The domains of these parameters are: 0 < E0x < 1, 0 < E0z < 1, 400 nm <  l < 700 nm, 0 < q < 180 degrees, and 0 < f< 360 degrees.

Unpolarized Incident Electric Field

When the incident field is unpolarized, there are no incident field parameters for the user to vary. The variables q and f may still be varied. Example 2 investigates the polarization of the scattered light when the incident light is unpolarized.
